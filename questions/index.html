نمونه سوالات

Web Basics

۱) پروتکل‌های پایه‌ای وب

۱) تفاوت HTTP و HTML چیست؟ جایگاه و کارکرد هر یک چیست؟

پاسخ:

۲) مدل‌های معماری وب

۱) هر یک از دو مدل Client-Server و Request-Response را توضیح دهید. نسبت این دو مدل چیست؟ مثلا آیا هر سیستم Request-Response الزاما Client-Server هم هست؟

پاسخ:

۳) نسل‌های برنامه‌های وب

اگر نسل‌های برنامه‌های وب را، برنامه‌های استاتیک، برنامه‌های پویا، و اپ‌های مدرن وب بدانیم؛ وجه تمایز هر یک از این نسل‌ها چیست؟

پاسخ:

۴) مرورگر به عنوان سیستم عامل

جایگاه مرورگرها در وب امروزی نسبت به سیستم عامل‌ها چیست؟

پاسخ:

۵) تکنولوژی‌های وب

منظور از تکنولوژی‌های وب چیست؟ دو نمونه از تکنولوژی‌های وب را نام برده و کارکرد آن را توضیح دهید.

پاسخ:

۶) تکنولوژی‌ها و کتابخانه‌ها

تفاوت تکنولوژی‌های وب و کتابخانه‌های وب چیست؟ به عبارت دیگر، تکنولوژی‌های وب چه ویژگی خاصی دارند که آن‌ها را از کتابخانه‌های معمول وب (مثلا JQuery) متمایز می‌کند؟

پاسخ:

۷) «سرعت» وب

مولفه‌های مهم سرعت وب/اینترنت را نام برده و توضیح دهید. منظور از bursty بودن ترافیک وب چیست؟ با توجه به این موضوع، جایگاه و اهمیت نسبی مولفه‌های سرعت وب را توضیح دهید.

پاسخ:

۸) «وب»

تفاوت وب و اینترنت چیست؟ دو نمونه سرویس دیگر مبتنی بر اینترنت را نام ببرید.

پاسخ:

۹) پروتکل‌های متنی

منظور از متنی بودن پروتکل HTTP چیست؟ یک نمونه پروتکل متنی دیگر را نام ببرید. آیا امکان انتقال تصاویر و ویدئوها با پروتکل HTTP ناقض متنی بودن آن است؟

پاسخ:

۱۰) محتوای پروتکل HTTP

سه بخش اصلی پیام‌ها (درخواست یا پاسخ) در پروتکل HTTP چیست؟ گرامر و کارکرد هر بخش را بیان کنید.

پاسخ:

۱۱) سرآیند

کاربرد سرآیندها (هدرها) در پروتکل HTTP چیست؟ به تفکیک پیام‌های درخواست و پاسخ توضیح دهید. کارکرد چند هدر را با ذکر مثال توضیح دهید.

پاسخ:

۱۲) زبان Markup

منظور از Markup Language چیست؟ تفاوت یک Markup Language با دو نمونه از این زبان‌ها را توضیح دهید.

پاسخ:

CSS

۱) قالب‌دهی مستطیل قرمز

در ... کد مناسبی بنویسید که ظاهر صفحه به صورت شکل زیر شود.

<html>
    <head>
        ...
    </head>
<body>
    <div style="margin: 0"></div>
</body>
</html>

پاسخ:

۲) کارکرد clearfix

هدف از کد زیر چیست؟ هر یک از اجزای آن را توضیح دهید.

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}

پاسخ:

۳) مفهوم Graceful Degradation

مفهم Graceful Degradation را با ذکر مثال توضیح دهید.

پاسخ:

۴) رندر

فرآیند نمایش داده شدن یک صفحه توسط مرورگر (رندر) را شرح دهید.

پاسخ:

۵) ترتیب رندر

ترتیب رندر اجزای مختلف صفحه (html و css و js) نسبت به هم چگونه است؟ پردازش کدام اجزا به صورت غیرهمزمان صورت می‌‌گیرد؟

پاسخ:

۶) انتخاب‌گرها

بخش‌های انتخاب‌گر css زیر را توضیح دهید. 

#body section.text.farsi:hover > a[href~="wikipedia.org"]

پاسخ:

۷) آبشار

منطق اولویت‌بندی قانون‌های css و پارامترهای دخیل در آن را توضیح دهید.

پاسخ:

۸) صفحات کوچک

کد CSSی بنویسید که پس‌زمینه‌ی صفحه را در دستگاه‌هایی که طول و عرض آن‌ها هر دو کمتر از 300px است را قرمز نماید.

پاسخ:

۹) مدل جعبه‌ای

مقادیر مختلف box-sizing و مدل جعبه‌ای در هر حالت را توضیح دهید.

پاسخ:

JavaScript

۱) مقادیر هیچ مقدار

مقادیر null و undefined را در JS توضیح داده و مقایسه کنید.

پاسخ:

۲) مشکل حلقه و closure

چرا این کد به درستی کار نمی‌کند؟ آن را اصلاح کنید.

for (var i=0; i<a.length; i++) {
  a[i].addEventListener('click', function() {
    console.log(i);
  });
}

پاسخ:

۳) کارکرد نقطه

عملیات lookup یک فیلد روی یک شی، مثلا o1.x به چه صورتی در JS انجام می‌شود؟ این موضوع چگونه باعث پویایی زبان JS شده است؟

پاسخ:

۴) ساخت پویای صفحه

۴) این صفحه را با کد JS بسازید. فرض کنید پس از بارگذاری کامل DOM، یک div با شناسه‌ی page در صفحه وجود دارد. 

Title            <-- <h1>, when clicked, color should become red
P1               <-- <p>
P2
...
P100

۶) تساوی

تفاوت == و === چیست؟ استفاده از کدام یک مطلوب‌تر است؟

پاسخ:

۷) متغیرهای global

نگذاشتن var پیش از تعریف یک متغیر چه تاثیری در scope آن دارد؟ روش دیگر (و استانداردتر) انجام این کار از درون یک تابع چیست؟ استفاده از متغیرهای global چه مشکلی در ادغام کدهای مختلف ایجاد خواهد کرد؟

پاسخ:

۸) ارث‌بری

مفهوم prototype و زنجیره‌ی پروتوتایپ‌ها را توضیح دهید. پیاده‌سازی مفاهیم کلاس و ارث‌بری در JS چگونه با استفاده از این امکان صورت می‌گیرد؟

پاسخ:

Python

۱) زبان پایتون

بخش‌های ۴ و ۵ و ۹ از آموزش سایت پایتون

پاسخ:

۲) دیشکنری گویا

با ارث‌بری از کلاس dict و ایجاد تغییرات لازم در توابع، یک دیکشنری درست کنید که در هر بار نوشته شده یا خوانده شدن کلیدها، عبارتی را print کنید.

پاسخ:

۳) پیاده‌سازی‌های پایتون

دو پیاده‌سازی مهم پایتون را نام برده و محدودیت‌ها و برتری‌های هر یک را توضیح دهید.

پاسخ:

۴) تعریف یک خطی

منظور از Comprehension برای list یا dict چیست؟ چند نمونه از آن را در پایتون با ذکر کد مثال توضیح دهید.
